# v59.54.1 — Challenge History

Adds Challenge History inside the Rewards Challenges section.

## Included

- Server response includes recent user challenge history.
- Completed, active, and missed challenge counts.
- History list with status, date range, category, spend vs limit, and HP target.
- Public-safe Copy proof text for sharing progress.
- Empty state when no previous challenges exist.

## Not changed

- Rewards/admin payout
- Wallet verification
- Supabase schema
- Universal Check scoring
- Daily Routine formula
- Mascot power logic
- Transaction history / PnL / scam labels
- Game mode
